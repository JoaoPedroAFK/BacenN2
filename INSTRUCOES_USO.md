# 🚀 Instruções de Uso Rápido - Sistema BACEN

## 📋 Primeiros Passos

### 1. Abrir o Sistema
- Abra o arquivo `index.html` no seu navegador
- O sistema carregará automaticamente na tela de Dashboard

### 2. Carregar Dados de Demonstração (Opcional)
- Clique no botão "📊 Dados Demo" no canto superior direito
- Confirme para carregar exemplos de reclamações
- A página será recarregada automaticamente

## ➕ Como Registrar uma Nova Reclamação

### Passo a Passo:
1. **Clique em "➕ Nova Reclamação"** na navegação
2. **Preencha os campos obrigatórios** (marcados com *):
   - Data da Reclamação
   - Data de Recebimento  
   - Responsável
   - Nome do Cliente
   - CPF (será formatado automaticamente)
   - Motivo Reduzido
   - Status da Reclamação

3. **Configure o Módulo de Contato**:
   - Marque os canais utilizados (Atendimento, Reclame Aqui, BACEN, Procon, N2)

4. **Adicione Protocolos Anteriores** (se houver):
   - Clique em "➕ Adicionar Protocolo"
   - Digite o número do protocolo
   - Use ❌ para remover protocolos

5. **Registre Tentativas de Contato**:
   - Clique em "➕ Adicionar Tentativa"
   - Preencha data/hora, resultado e observações
   - Use ❌ para remover tentativas

6. **Clique em "💾 Salvar Reclamação"**

## 📋 Como Visualizar Reclamações

### Na Lista de Reclamações:
1. **Clique em "📋 Lista de Reclamações"**
2. **Use a busca** para filtrar por:
   - Nome do cliente
   - CPF
   - Motivo da reclamação

3. **Use o filtro de status** para ver apenas:
   - Não Iniciado
   - Em Tratativa
   - Concluído
   - Respondido

4. **Clique nos cards** para expandir e ver todos os detalhes

## 📊 Como Gerar Relatórios

### Tipos de Relatórios Disponíveis:

#### 1. Relatório por Período
- Clique em "📈 Relatórios"
- Clique em "Gerar Relatório" no card "Relatório por Período"
- Digite a data inicial (DD/MM/AAAA)
- Digite a data final (DD/MM/AAAA)
- Visualize o relatório gerado

#### 2. Relatório por Status
- Clique em "Gerar Relatório" no card "Relatório por Status"
- Veja a distribuição percentual por status
- Gráfico visual com barras de progresso

#### 3. Relatório por Responsável
- Clique em "Gerar Relatório" no card "Relatório por Responsável"
- Veja a performance de cada responsável
- Quantidade e percentual de reclamações

#### 4. Relatório Completo
- Clique em "Gerar Relatório" no card "Relatório Completo"
- Tabela com todas as reclamações
- Dados completos para análise

### Exportar Relatórios:
- Após gerar qualquer relatório, clique em "📊 Exportar Relatório"
- O arquivo CSV será baixado automaticamente

## 🆘 Solução de Problemas

### Se o sistema não carregar:
- Verifique se todos os arquivos estão na mesma pasta
- Abra o console do navegador (F12) para ver erros
- Certifique-se de que o JavaScript está habilitado

### Se os dados não salvarem:
- Verifique se o localStorage está habilitado
- Tente limpar o cache do navegador
- Use o botão "🗑️ Limpar Dados" para resetar

### Se houver erro de validação:
- Verifique se todos os campos obrigatórios estão preenchidos
- Confirme se o CPF está correto
- Verifique se as datas estão no formato correto

## 📞 Campos Importantes

### Obrigatórios (marcados com *):
- Data da Reclamação
- Data de Recebimento
- Responsável
- Nome do Cliente
- CPF
- Motivo Reduzido
- Status da Reclamação

### Opcionais:
- Prazo Retorno BACEN
- Origem
- Motivo Detalhado
- Telefone
- Histórico de Tentativas
- Módulo de Contato
- Protocolos Anteriores
- Resultado da Tratativa

## 🎯 Status das Reclamações

1. **Não Iniciado**: Reclamação registrada, aguardando início
2. **Em Tratativa**: Sendo analisada e tratada
3. **Concluído**: Tratamento finalizado com resolução
4. **Respondido**: Resposta enviada ao cliente
